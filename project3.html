💡 Advanced Beginner

📋 Modern Signup Form UI

Project introduction: Design a beautiful, fully functional form with floating labels, real-time validation styling, and a sleek layout. Perfect for learning advanced CSS pseudo-classes and form accessibility.

🎯 Learning objectives

  • Style form elements consistently across browsers.
  • Use :focus, :valid, :invalid pseudo-classes.
  • Implement custom checkbox/radio styles.
  • Create a responsive two-column form layout.

🛠️ Skills covered

  • Advanced CSS selectors, transitions, shadows, and gradients.
  • Form accessibility (labels, fieldsets).
  • Responsive input groups.

⚡ Challenge section

Add custom error messages using only CSS (hint: :placeholder-shown or adjacent sibling). Style the submit button with a gradient that moves on hover.

📝 Student tasks section

  1. Build a signup form with: full name, email, password, confirm password, terms checkbox.
  2. Apply modern border, focus glow effect.
  3. Use flex/grid to align labels and inputs.
  4. Make the form responsive (stack on mobile).
  5. Add a success-style feedback for valid inputs.
  6. Create a “Submit” button with hover animation.

💡 Tips section

  • Use `outline: none` but replace with visible focus ring for accessibility.
  • Combine `:focus-within` to style entire field groups.
  • Test different input states: required, pattern.
  • Keep color contrast high.
