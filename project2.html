🚀 Intermediate Beginner

🖼️ Responsive Image Gallery

Project introduction: Build a fully responsive photo gallery that adapts beautifully from desktop to mobile using CSS Grid and Flexbox. Showcase images with captions and smooth hover effects.

🎯 Learning objectives

  • Implement CSS Grid layout for gallery cards.
  • Use object-fit for consistent image sizing.
  • Apply media queries to rearrange columns.
  • Create interactive hover overlays.

🛠️ Skills covered

  • CSS Grid & Flexbox combination, responsive units (fr, %, vw).
  • Pseudo-classes :hover, transition properties.
  • Responsive images and accessibility.
🖼️ FINAL PREVIEW IMAGE PLACEHOLDER 🖼️
(Gallery with 6 cards – 3 columns on desktop, 2 on tablet, 1 on mobile with zoom effect)

⚡ Challenge section

Add a lightbox feature (JavaScript optional for advanced) or create a “lazy loading” mockup. Make the captions slide up on hover.

📝 Student tasks section

  1. Create a container with class `.gallery`.
  2. Add at least 6 image cards (you can use placeholder images via unsplash or local).
  3. Each card must have an image, a title, and a short description.
  4. Make the gallery responsive: desktop 3 cols, tablet 2 cols, mobile 1 col.
  5. Add a hover effect: scale or shadow.
  6. Ensure images don’t distort.

💡 Tips section

  • Use `grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));` for easy responsiveness.
  • Set `max-width: 100%; height: auto;` for images.
  • Use `transition: transform 0.2s` for smooth scaling.
  • Test on real devices or devtools.
