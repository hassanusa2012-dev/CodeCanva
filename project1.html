🏁 Beginner Project

📄 Personal Bio Page

Project introduction: Create your first complete HTML/CSS webpage – a personal bio card that introduces you (or a fictional person) with profile image, about section, hobbies, and social links.

🎯 Learning objectives

  • Understand basic HTML structure (html, head, body, semantic tags).
  • Apply CSS properties: colors, fonts, margins, padding, borders.
  • Build a simple two-column layout using Flexbox.
  • Add hover effects and basic responsiveness.

🛠️ Skills covered

  • HTML5: headings, paragraphs, divs, spans, lists, images.
  • CSS: selectors, box model, background, border-radius, Flexbox basics.
  • Google Fonts integration.
✨ FINAL PREVIEW IMAGE PLACEHOLDER ✨
(Your bio page will look like a modern profile card with a clean avatar, description, and styled buttons)

⚡ Challenge section

Make your bio page stand out: use custom gradient background, round avatar, and create a "fun fact" section that flips color on hover.

📝 Student tasks section

  1. Create a new HTML file and link a CSS file.
  2. Add a header with name + tagline.
  3. Insert a placeholder image (or Font Awesome icon).
  4. Write a short bio paragraph.
  5. List 3 hobbies or skills.
  6. Add two buttons: "Contact" and "Portfolio".
  7. Make the page responsive (mobile friendly).

💡 Tips section

  • Use CSS variables for easy theming.
  • Always add `alt` text for images.
  • Test your page in Chrome DevTools mobile view.
  • Don't be afraid to use `border-radius: 999px` for circular avatar.
